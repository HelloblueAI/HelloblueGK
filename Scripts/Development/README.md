# Development Scripts

This directory contains utility scripts for local development and testing.

## Scripts

- **`run-demo.sh`** - Run the Plasticity demo locally
- **`find-dotnet.sh`** - Locate .NET installation on the system

## Usage

### Run Demo
```bash
./run-demo.sh
```

### Find .NET
```bash
./find-dotnet.sh
```
