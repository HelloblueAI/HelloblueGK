## Asset Guidelines

### Images
- Use descriptive filenames
- Prefer PNG for diagrams and technical illustrations
- Prefer SVG for scalable graphics when possible
- Keep images optimized for documentation purposes

### Organization
- General/promotional images ‚Üí `Assets/Images/`
- Design-specific images ‚Üí `Docs/Designs/{design-name}/`
- Component visualizations ‚Üí Generated from scripts in `Scripts/Visualization/`
